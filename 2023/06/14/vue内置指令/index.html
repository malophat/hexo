v-for

v-for=”item in items”

分情况：

  • 如果items是[{}，{}]，那么item就是{}，要通过item.键名 来调用值

  • 如果items是{}，那么item就是{}里的值，

    如果items是整数，那么item就是[1,items]

v-if

v-if=”表达式” 如果表达式是true则显示该标签，false则不显示

v-if与v-else搭配使用，一个不显示则另外一个显示

v-model

v-model用于input标签，让Input中输入（或选取）的值，与data里的数据同步，

data：{

​ message：”content”，//如果标签绑定：v-model=”message”,那么input里的内容也会是content。

}

如果intput[type=”radio”]那么会将input设置的value值传输给message

如果input[type=”checkbox”]并且可多选，那么应将message：[]设为空数组来接收

v-if=”表达式”，与 v-else配套使用（作用：是否显示该标签）

v-bind

v-bind：用于绑定属性和样式

  1. 带有{}

    只设置一个属性

    v-bind：class=”{name：istrue}”

    data:{

    ​ istrue：true则绑定class=name属性值，false则失败

    }

    同时设置多个属性

    v-bind：class=”{name：istrue，secondName：isset}”

    data:{

    ​ istrue：true则绑定class=name属性值，false则失败

    ​ isset：true则绑定class=secondName属性值，false则失败

    }

  2. 不带{}

    只设置一个属性 就一个属性对应一个值

    这里class可以替换成value或id等，值就可以绑定data里面的数据了

v-bind：class=”name”

data:{

​ name：”str”

}

#class=”str”

同时设置多个属性 封装到一个对象里

v-bind：class=”obj”

data:{

​ obj：{

​ name：true则绑定class=name属性值，false则失败

​ secondName：true则绑定class=secondName属性值，false则失败

​ }

}

v-bind：设置样式

b-bind：style=”obj”

data：{

​ obj：{

​ width：”20px”，

​ height：”20px”，

​ backgroundcolor：”red”

​ }

}
